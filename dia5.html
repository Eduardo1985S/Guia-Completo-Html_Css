🔗 Revisão: CSS Externo (link)

Reforce a base: como conectar e organizar folhas de estilo

🤔 Por que usar CSS externo?

O CSS externo separa conteúdo (HTML) de apresentação (CSS), facilitando manutenção, reutilização e organização do código.

📋 3 Formas de Aplicar CSS:

❌ CSS Inline (Não recomendado)
<p style="color: blue; font-size: 18px;">
    Texto com CSS inline
</p>
Problemas:
  • Dificulta manutenção
  • Não reutilizável
  • Código HTML poluído
  • Dificulta trabalho em equipe
⚠️ CSS Interno (Uso limitado)
<head>
    <style>
        p {
            color: blue;
            font-size: 18px;
        }
    </style>
</head>
Quando usar:
  • Páginas únicas
  • Testes rápidos
  • Estilos muito específicos
✅ CSS Externo (Recomendado)
<head>
    <link rel="stylesheet" href="styles.css">
</head>
Vantagens:
  • Reutilizável em múltiplas páginas
  • Fácil manutenção
  • Melhor organização
  • Cache do navegador
  • Trabalho em equipe facilitado

📁 Organização Recomendada:

📁 projeto/
📄 index.html
📄 sobre.html
📄 contato.html
📁 assets/
📁 css/
📄 style.css
📄 responsive.css
📁 images/
📁 js/
💡 Dicas de Organização:
  • assets/ - todos os recursos (CSS, imagens, JS)
  • css/ - apenas arquivos de estilo
  • style.css - estilos principais
  • responsive.css - media queries
  • Use nomes descritivos para os arquivos

🔗 Conectando Múltiplos Arquivos CSS:

index.html VS Code - Dracula Theme 
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Meu Site</title>
    
    <!-- CSS Reset/Normalize -->
    <link rel="stylesheet" href="assets/css/reset.css">
    
    <!-- CSS Principal -->
    <link rel="stylesheet" href="assets/css/style.css">
    
    <!-- CSS Responsivo -->
    <link rel="stylesheet" href="assets/css/responsive.css">
    
    <!-- CSS de Componentes -->
    <link rel="stylesheet" href="assets/css/components.css">
</head>
⚠️ Ordem Importa!

Os arquivos CSS são aplicados na ordem que aparecem. O último arquivo pode sobrescrever regras dos anteriores.

1 reset.css - Remove estilos padrão
2 style.css - Estilos principais
3 responsive.css - Adaptações mobile
4 components.css - Componentes específicos

🎯 Seletores CSS: tag, classe, id, universal

Domine as formas de selecionar e estilizar elementos HTML

🎯 O que são Seletores?

Seletores são padrões que identificam quais elementos HTML devem receber determinado estilo. É como dizer ao CSS: "aplique esta formatação nestes elementos específicos".

🏆 Hierarquia de Especificidade:

👑
Estilo Inline

style="color: red;"

Especificidade: 1000
🆔
ID (#)

#meuId

Especificidade: 100
🏷️
Classe (.)

.minhaClasse

Especificidade: 10
🏷️
Tag

p, div, h1

Especificidade: 1

🏷️ Seletor de Tag (Elemento):

Seleciona todos os elementos de um tipo específico

CSS:
p {
    color: #2c3e50;
    font-size: 16px;
    line-height: 1.5;
}

h1 {
    color: #e74c3c;
    font-size: 32px;
    font-weight: bold;
}

a {
    color: #3498db;
    text-decoration: none;
}
HTML:
<h1>Título Principal</h1>
<p>Este parágrafo será estilizado.</p>
<p>Este também será estilizado.</p>
<a href="#">Link estilizado</a>
📋 Resultado:

Título Principal

Este parágrafo será estilizado.

Este também será estilizado.

Link estilizado
📝 Características:
  • Afeta todos os elementos da tag
  • Baixa especificidade (1)
  • Ideal para estilos base/globais
  • Fácil de sobrescrever

🏷️ Seletor de Classe (.):

Seleciona elementos que possuem o atributo class especificado

CSS:
.destaque {
    background: #f1c40f;
    padding: 10px;
    border-radius: 5px;
    font-weight: bold;
}

.botao {
    background: #3498db;
    color: white;
    padding: 12px 24px;
    border: none;
    border-radius: 6px;
    cursor: pointer;
}

.texto-pequeno {
    font-size: 12px;
    color: #7f8c8d;
}
HTML:
<p>Parágrafo normal</p>
<p class="destaque">Parágrafo destacado</p>
<button class="botao">Meu Botão</button>
<span class="texto-pequeno">Texto pequeno</span>
<p class="destaque">Outro destaque</p>
📋 Resultado:

Parágrafo normal

Parágrafo destacado


Texto pequeno

Outro destaque

📝 Características:
  • Reutilizável em múltiplos elementos
  • Especificidade média (10)
  • Mais usado no desenvolvimento
  • Permite múltiplas classes no mesmo elemento

🆔 Seletor de ID (#):

Seleciona o elemento com o atributo id específico. Deve ser único na página

CSS:
#cabecalho {
    background: #2c3e50;
    color: white;
    padding: 20px;
    text-align: center;
}

#rodape {
    background: #34495e;
    color: white;
    padding: 15px;
    text-align: center;
    margin-top: 50px;
}

#menu-principal {
    background: #3498db;
    padding: 10px;
}
HTML:
<header id="cabecalho">
    <h1>Meu Site</h1>
</header>

<nav id="menu-principal">
    Menu de navegação
</nav>

<main>
    Conteúdo principal
</main>

<footer id="rodape">
    © 2024 Meu Site
</footer>
📋 Resultado:

Meu Site

Menu de navegação
Conteúdo principal
© 2024 Meu Site
📝 Características:
  • Único por página
  • Alta especificidade (100)
  • Usado para elementos únicos
  • Ideal para JavaScript também
  • Difícil de sobrescrever

🌍 Seletor Universal (*):

Seleciona todos os elementos da página

CSS:
/* Reset básico */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

/* Ou fonte padrão */
* {
    font-family: Arial, sans-serif;
}

/* Debug - ver todos elementos */
* {
    border: 1px solid red;
}
HTML:
<h1>Título</h1>
<p>Parágrafo</p>
<div>
    <span>Texto</span>
    <a href="#">Link</a>
</div>
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
🎯 Usos Comuns:
🔄 CSS Reset
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

Remove margens e paddings padrão

🔤 Fonte Global
* {
    font-family: 'Roboto', Arial, sans-serif;
}

Define fonte para todos elementos

🐛 Debug
* {
    border: 1px solid red;
    background: rgba(255,0,0,0.1);
}

Visualizar layout durante desenvolvimento

⚠️ Cuidados:
  • Pode impactar performance
  • Use com moderação
  • Ideal apenas para reset ou configurações globais
  • Especificidade muito baixa (0)
