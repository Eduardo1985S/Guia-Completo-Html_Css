O Box Model do CSS

O Box Model é um dos conceitos mais importantes do CSS. Todo elemento HTML é tratado como uma "caixa" retangular com 4 componentes principais:

Visualização Interativa do Box Model

CONTENT

Conteúdo do elemento

150px × 100px

CONTENT

O conteúdo real: texto, imagens, etc.

Propriedades: width, height

PADDING

Espaço interno entre conteúdo e borda

Propriedades: padding, padding-top, padding-right, padding-bottom, padding-left

BORDER

Linha ao redor do padding e conteúdo

Propriedades: border, border-width, border-style, border-color

MARGIN

Espaço externo entre elementos

Propriedades: margin, margin-top, margin-right, margin-bottom, margin-left

Box-Sizing: Como o Tamanho é Calculado

content-box (padrão)

width/height = apenas o conteúdo

.elemento { width: 200px; height: 100px; padding: 20px; border: 5px solid #333; /* Tamanho final: 250px × 150px */ }
200px × 100px + padding + border = 250px × 150px

border-box (recomendado)

width/height = total (incluindo padding e border)

.elemento { box-sizing: border-box; width: 200px; height: 100px; padding: 20px; border: 5px solid #333; /* Tamanho final: 200px × 100px */ }
200px × 100px tamanho total fixo

Melhor Prática

É recomendado usar box-sizing: border-box em todos os elementos:

/* No início do seu CSS */ * { box-sizing: border-box; } /* Ou mais específico */ html { box-sizing: border-box; } *, *::before, *::after { box-sizing: inherit; }

Margin e Padding em Detalhes

Sintaxes de Margin e Padding

1 Valor
margin: 20px;

Aplica 20px em todos os lados

2 Valores
margin: 20px 10px;

20px (vertical) / 10px (horizontal)

3 Valores
margin: 20px 10px 5px;

20px (top) / 10px (horizontal) / 5px (bottom)

4 Valores
margin: 20px 15px 10px 5px;

Sentido horário: top, right, bottom, left

Margin Collapse (Colapso de Margens)

Quando duas margens verticais se encontram, elas "colapsam" em uma única margem:

Elemento 1

margin-bottom: 30px
Espaço resultante: 30px (não 50px!)

Elemento 2

margin-top: 20px
.elemento1 { margin-bottom: 30px; } .elemento2 { margin-top: 20px; } /* Resultado: apenas 30px de espaço entre eles */ /* (a maior margem prevalece) */

Centralizando Elementos

Centralização Horizontal
.elemento { width: 300px; margin: 0 auto; /* top/bottom: 0, left/right: auto */ }
Centralizado
Margin Negativa (Técnica Avançada)
.elemento { position: relative; left: 50%; margin-left: -150px; /* metade da largura */ }

💡 Útil para elementos com posição absoluta

Containers e Layout com DIV

O Elemento DIV - Container Universal

O elemento <div> é um container genérico usado para agrupar outros elementos e criar layouts.

Sem Semântica

Não possui significado específico, apenas agrupa conteúdo

Block-level

Ocupa toda a largura disponível por padrão

Totalmente Estilizável

Pode receber qualquer estilo CSS

Aninhável

Pode conter outros divs e elementos
